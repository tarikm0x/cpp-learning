# cpp-learning

C++ practice problems and exercises following the Programming I & II
curriculum at the Faculty of Information Technologies, Mostar.

## Programming I
- `01-osnove/` - Basics, data types, I/O
- `02-operatori-izrazi/` - Operators and expressions
- `03-kontrolne-strukture/` - Control structures
- `04-petlje/` - Loops
- `05-nizovi-staticki/` - Static arrays
- `06-pokazivaci-dinamicki/` - Pointers and dynamic memory
- `07-funkcije/` - Functions
- `08-stringovi/` - Strings
- `09-rekurzija/` - Recursion
- `10-strukture/` - Structures

## Programming II
- `11-OOP-klase/` - OOP, classes and objects
- `12-specijalni-clanovi/` - Special class members
- `13-genericke-klase/` - Generic functions and classes
- `14-asocijacije/` - Associations, composition, references
- `15-nasljedjivanje/` - Inheritance
- `16-apstraktne-klase/` - Abstract classes
- `17-polimorfizam/` - Polymorphism
- `18-izuzeci/` - Exceptions
- `19-STL/` - Standard C++ library
- `20-fajlovi/` - File handling
- `21-multithreading/` - Multi-threading
